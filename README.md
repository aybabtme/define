# Define

CLI tool to get definitions.

# Usage

```bash
$ define concurrency
concurrency
Web definitions
concurrence: agreement of results or opinions
http://wordnetweb.princeton.edu/perl/webwn?s=concurrency

$ define extreme prejudice
extreme prejudice
Web definitions
In military and other covert operations, terminate with extreme prejudice is a euphemism for execution (playing on the expression "termination with prejudice" of an employment contract). In a military intelligence context, it is generally understood as an order to assassinate. ...
http://en.wikipedia.org/wiki/Extreme_prejudice
```
